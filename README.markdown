COI / JBoss
===

Instalation and management of JBoss Aplication Server and JBoss Enterprise Application Platform.

Supports:
---

* Installs JBoss AS or JBoss EAP
* Installs required JDK (defaults: JDK 7)
* Manage instalation parametrers
* Management of: 
    * datasources, 
    * deployments, 
    * jms queues, 
    * security domains
